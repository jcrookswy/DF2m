This project is for the FH146, an ultra low cost angle of arrival direction finder for 2 meter transmitter hunts, based on the Raspberry Pi Pico 2. 
It is built in Visual Studio 2019, and uses PortAudio, WxWidgets, and Intel's Integrated Performance Primitives. 
While it will not currently compile for Linux, it shouldn't be too much work to convert it.
Included is rev B1 firmware for the Pico2.
